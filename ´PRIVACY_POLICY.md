# Privacy Policy

Last updated: 2026

## Overview

Habits does not collect, store, or transmit any personal data.

All habit data, reward data, and logs are stored locally on the user's device.

## Data Collection

Habits does not collect any personal information.

The app does not require user accounts or cloud services.

## Local Storage

All application data is stored locally on the user's device using the Android local database system.

No data is transmitted to external servers.

## Analytics and Tracking

Habits does not use:

- Analytics services
- Advertising networks
- Tracking technologies

## Third-Party Services

Habits does not integrate with any third-party services that collect user data.

## Changes to This Policy

If the privacy policy changes in future versions of the app, updates will be published in this repository.

## Contact

For questions about this privacy policy, please open an issue in the GitHub repository.
